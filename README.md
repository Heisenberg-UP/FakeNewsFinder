**Install Guide:**
1. Run the following command:
   ```bash
   git clone https://github.com/Heisenberg-UP/FakeNewsFinder
   cd FakeNewsFinder/
   python3 -m venv env
   source env/bin/activate
   pip install -r files/requirements.txt
   deactivate
   ```

3. Test install by running the following command:
   ```bash
   bash run.sh
   ```
